# Sentinel Ops

Sentinel Ops es una aplicación full stack premium para una central de monitoreo operativo en tiempo real.

## Stack

- React + Vite
- Framer Motion
- Tailwind-ready CSS tokens y utilidades visuales premium
- Node.js + Express
- Firebase Firestore ready para listeners realtime

## Ejecutar

```bash
npm install
npm run dev
npm run server
```

Copiá `.env.example` a `.env` y completá las credenciales de Firebase para activar Firestore real.
